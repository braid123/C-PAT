函数申明： int lock(int handle, long offset, long length)
函数用途： 对文件中的某个区域进行锁定，禁止其他程序操作该区域，用于支持网络环境下的文件共享操作
头 文 件： io.h
输入参数： 
handle：文件句柄；offset：文件区域的起始位置；length：文件区域的长度
输出参数：
返 回 值： 0：锁定成功；非0：锁定失败
使用说明：
相关函数： | unlock |

函数申明： long lseek(int handle, long offset, int fromwhere)
函数用途： 移动文件位置指针到指定位置
头 文 件： io.h
输入参数： 
handle：文件句柄；offset：文件位置；fromwhere：从文件的何处开始移动，该参数可使用以下3个宏定义：
fromwhere宏定义  数值   含义
SEEK_CUR	1        当前位置计算offset
SEEK_END	2        文件结束位置计算offset,此时offset为负数 
SEEK_SET	0        文件开始位置计算offset
输出参数：
返 回 值：
使用说明：
相关函数： | fseek | open | read | write | close |

函数申明： struct tm *localtime(const time_t *timer)
函数用途： 返回一个以tm结构表达的机器时间信息
头 文 件： time.h
输入参数： 
timer:使用time()函数获得的机器时间,time_t实际上是一个long类型参数，见文件time.h
输出参数：
返 回 值： 
以tm结构表达的时间，结构tm定义如下：  
	struct	tm	{
		int	tm_sec;   
		int	tm_min;    
		int	tm_hour;
		int	tm_mday;
		int	tm_mon;
		int	tm_year;
		int	tm_wday;
		int	tm_yday;
		int	tm_isdst;
	};
使用说明：
相关函数： | time |
