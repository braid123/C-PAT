函数申明： int access(char *filename , int mode)
函数用途： 文件状态检查
头文件： io.h
输入参数： finename:文件名称；mode: 模式，共5个输入值：
0-检查文件是否存在
1-检查文件是否可运行
2-检查文件是否可写访问
4-检查文件是否可读访问
6-检查文件是否可读/写访问
输出参数： 0：允许访问，-1：禁止访问
使用说明： UNIX文件函数，不是ANSIC C标准定义
相关函数： chmod

函数申明： char *asctime(struct tm *ptr)
函数用途： 得到机器时间
头 文 件： time.h
输入参数： 结构指针ptr应通过函数localtime()和gmtime()得到(如下例)
输出参数：
返 回 值：如下格式的时间字符串指针：星期 月 日 小时:分：秒 年\n\0
使用说明： 调用举例如下： 
#include<time.h>
#include<stddef.h>
main( )
{
struct tm *ptr;
time_t lt;

    lt=time(NULL);
    ptr=locatime(<);
    printf("%s current time is:",asctime(ptr));
}
相关函数： localtime | time | ctime | gmtime |
