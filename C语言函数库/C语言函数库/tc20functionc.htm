函数申明： int creat(char *filename,  int pmode) 


函数用途：  创建一个文件 


头 文 件： io.h


输入参数： 

filename:文件名称；
pmode:  创建方式，取值和含义如下：
  S_IFMT:   file type mask        S_IFREG:	regular  
  S_IFDIR:  directory             S_IREAD:	owner may read 
  S_IFIFO:  FIFO special 	  S_IWRITE:	owner may write 
  S_IFCHR:  character special     S_IEXEC:	owner may execute  
  S_IFBLK:  block special 





输出参数：  


返 回 值：－1:创建文件失败,其他数值表示创建文件的句柄号
