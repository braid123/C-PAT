函数申明： int ungetc(int c, FILE *fp)
函数用途： 将字符c放到文件流fp的首部，下一次流输入函数（如getc）函数将首先读到该字符
头 文 件： stdio.h
输入参数： c:要写入的字符；fp:文件流指针
输出参数：
返 回 值： 返回字符c,操作失败返回EOF
使用说明：
