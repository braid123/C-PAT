函数申明： int bioscom(int cmd, char byte, int port)
函数用途： RS232串口通信口操作
头 文 件： bios.h
输入参数： 
cmd: 0-初始化串口,1-发送一个字符,2-接受一个字符,3:返回串口状态
byte: 波特率和发送位数设置，各位的含义如下：
  B7,B6,B5:波特率设置，具体设置为：9600波特－111,4800－110,2400－101,1200－100,600－011,300－010
  B4,B3   :奇偶校验设置,具体设置为：00或10－无奇偶，01－奇校验，11－偶校验。
  B2      :停止位个数，1－2个停止位，0－1个停止位
  B1,B0   :10-7个数据位; 11-8个数据位
port:串口号,0,1,2,3等
输出参数：
返 回 值： 
16位的串口状态信息，各位的含义分别如下：
  高8位                      低8位(cmd=0,1,3时)        当cmd=2时，低8位用于保存接收的数据   
  B15: 超时错误               B7: 检测到线路信号
  B14: 发送移位寄存器空       B6: 振铃指示
  B13: 发送保持寄存器空       B5: 数据装置就绪
  B12: 中止检测错误           B4: 数据发送结束 
  B11: 帧错误                 B3: 线路信号变化 
  B10: 奇偶错误               B2: 脉冲后沿振铃检测  
  B9 : 溢出错误               B1: 数据装置就绪变化
  B8 : 数据准备就绪           B0: 数据发送结束变化

函数申明： int bioskey(int cmd)
函数用途： 键盘值读取
头 文 件： bios.h
输入参数： 
cmd=0: 返回一个键盘值，如无键盘按下，一直等待
cmd=1: 查询键盘是否按下。返回0，表示无键按下；返回非0，表示有键按下
cmd=2: 返回控制键状态，返回值保存在低8位中，含义如下：
        B0: 右边的shift键按下    B5: 已打开Scroll Lock
        B1: 左边的shift键按下    B6: 已打开Num Lock
        B3: Ctrl键按下           B7: 已打开Caps Lock
        B4: Alt键按下            B8: 已打开Insert
输出参数：
返 回 值： 含义见上
使用说明：
相关函数： getche | kbhit |

函数申明： int biosprint(int cmd , int byte ,int port)
函数用途： 打印口控制
头 文 件： bios.h
输入参数： 
cmd:  0-打印byte中的字符；1-初始化打印口状态；2-返回打印口状态。
byte: 当cmd=1时，为要打印的字符。
port: 打印口，0－LPT1;  1-LPT2
输出参数：
返 回 值： 
当cmd=2时，各位含义如下：
     B0:        超时错误   B5:  已联机         B8: 打印机闲
     B1,B2,B3:  未使用     B6:  打印纸用完     B9,B10,B11,B12,B13,B14,B15: 未使用 
     B4:        I/O错误    B7:  认可
使用说明：
相关函数： bioscom

函数申明： long biostime( int cmd , long newtime)
函数用途： 设置和读取BIOS时钟
头 文 件： bios.h
输入参数： 
cmd: 0－读取时钟；1－设置时钟
newtime：当cmd=1时，为新的设置数值
输出参数：
返 回 值：cmd=0时，返回时钟值
使用说明：该时钟为一个每秒18.2次的计数累计值，并在每日24时清零。
相关函数： time | ctime
